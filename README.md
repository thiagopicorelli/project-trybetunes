# Trybetunes
Trabalho feito na Trybe de uma aplicação que permite você buscar por artistas e escutar um preview de suas músicas. Ele também permite que o usuário favorite suas músicas preferidas e também crie seu próprio perfil na aplicação. Testado por arquivos de teste do RTL e ESLint.
